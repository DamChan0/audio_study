# 들어가며

이 문서는 `RuStudio Phase 2 - DSP 기초`를 순차적으로 학습하기 위한 mdBook이다.

`01_cpal`이 "오디오 장치에 샘플을 내보내는 법"을 다뤘다면, 여기서는 **그 샘플을 어떻게 만들고/바꾸고/측정할 것인가**를 다룬다.

```text
cpal이 준 버퍼를 어떻게 채울 것인가?
-> 오실레이터로 기본 파형 만들기
-> 엔벨로프로 시간축 컨트롤
-> 필터로 주파수축 컨트롤
-> FFT로 주파수축 분석
-> LUFS로 라우드니스 측정
```

이 책의 목표는 단순 API 사용법이 아니라 **DSP 블록을 직접 구현해보면서 수식과 코드의 간격을 좁히는 것**이다.

## 이 책을 다 읽고 나면

- 오실레이터의 위상 누산이 왜 `f64`가 더 안전한지 설명할 수 있다.
- ADSR이 왜 상태머신으로 구현되는지 설명할 수 있다.
- Biquad 계수가 바뀔 때 클릭 노이즈가 왜 생기는지 설명할 수 있다.
- RBJ cookbook의 peaking 계수 공식이 어떻게 동작하는지 읽을 수 있다.
- FFT 프레임 크기와 hop size가 스펙트럼 해상도/반응성에 주는 영향을 설명할 수 있다.
- K-weighting 2단 Biquad의 역할을 설명할 수 있다.

## 추천 읽기 흐름

1. `이 책을 읽는 방법`
2. `DSP가 cpal 위에서 푸는 문제`
3. `오실레이터와 위상 누산`
4. `ADSR 엔벨로프`
5. `앨리어싱과 PolyBLEP`
6. `Biquad 필터` → `RBJ Cookbook / EQ`
7. `FFT / STFT`
8. `LUFS (BS.1770)`
9. `복습` → `다음 단계`

## 이 책과 예제의 관계

- `src/*.md`: 개념 / 질문 / 완료 기준
- `examples/src/oscillator/*`, `examples/src/filter/*` …: 직접 구현할 DSP 블록 스텁
- `examples/src/bin/*`: 각 블록을 `cpal` 스트림에 연결해 소리로 확인하는 진입점

각 챕터는 "읽기 → 질문에 답 → 예제 스텁 구현 → 소리로 검증" 순서로 읽는다.
